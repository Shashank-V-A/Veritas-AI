# Veritas AI

> **Don't consume information. Verify it.**

An AI-powered credibility analysis platform that evaluates news, social posts, transcripts, and more — producing professional trust reports instead of chat responses.

## Tech Stack

- **Frontend:** React, Vite, TypeScript, TailwindCSS, shadcn/ui, Framer Motion, Recharts
- **Backend:** Node.js, Express (Vercel serverless)
- **Database:** SQLite + Prisma
- **AI:** Mesh API (exclusive provider)
